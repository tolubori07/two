As we all know, sometimes writing code can be a bit of a pain in the backside when things get complicated...

So to save my sanity I use some libraries to make my tasks easier...



  • Jquery: (I don't use it frequently as it is widely regarded as obsolete nowadays, but I still learnt it as it was a great library to use that made writing code easier) Makes writing javascript code easier with its inbuilt functions. It's capable of shortening the amount of code written to just a Third. However, speed became an issue as it was 40× slower than vanillaJs.

  • Bootstrap (wasn't used in building this site): A css/js library that makes styling websites easier by using convenient class names on elements.

  • ThreeJs:A Javascript library that makes 3D designs and animations easier using webGL

  • AnimeJs: A javascript library that makes staggering animations easier.
